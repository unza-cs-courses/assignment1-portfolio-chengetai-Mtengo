Design Theme:
The design theme of this portfolio focuses on agriculture, landscaping, and project management. The goal of the website is to present projects related to landscape design, agricultural planning, and sustainable outdoor development in a clean and professional way. The color palette uses natural green tones and light backgrounds to represent nature, growth, and environmental sustainability. The layout is simple and structured so that visitors can easily navigate between the Home, About, Projects, and Contact sections. The overall style emphasizes clarity, readability, and a modern responsive design that works across different devices.

Challenges & Solutions

One of the challenges I faced during this assignment was making the layout responsive across different screen sizes. Initially the project cards and navigation menu did not display properly on smaller screens. To solve this, I implemented a mobile-first approach and used CSS Grid to control the layout of the project cards. I also added media queries at 768px and 1024px to adjust the number of columns and spacing for tablet and desktop screens.

Another challenge was organizing the navigation menu for mobile devices. I solved this by implementing a CSS-only hamburger menu using a checkbox toggle and CSS selectors. This allowed the navigation links to be hidden on small screens and displayed when the menu icon is clicked.

I also experimented with CSS animations and transitions to improve the user experience. For example, the hero section fades in when the page loads and project cards have a hover effect that slightly lifts them to make the interface more interactive.
